# LaueSim

銅酸化物の c 軸方向から X 線を入射し、入射側に戻ってくる後方反射 Laue スポットをブラウザ上でシミュレートする単体HTMLアプリです。

## 使い方

`index.html` をブラウザで開いてください。依存パッケージやビルド手順は不要です。

入力できる条件:

- スクリーンまでの距離 `L` mm
- 格子定数 `a,b,c` Å
- 入射方向 `[u v w]`
- 入射方向/試料座標の回転角 `a軸,b軸,c軸` deg
- X線スペクトルの波長範囲
- 探索する最大ミラー指数

グラフ操作:

- ドラッグ: 表示範囲を移動
- マウスホイール/トラックパッドスクロール: 拡大縮小
- 照準ボタン: 表示をリセット

現在のモデルは直交格子の後方反射 Laue です。初期値は `a=b=3.82 Å, c=13.2 Å`、入射方向 `[0 0 1]` で、層状銅酸化物の c 軸入射を想定しています。

各逆格子ベクトル `(h/a, k/b, l/c)` に対して Laue 条件を満たす波長を計算し、回折方向が入射方向と反対側へ戻るスポットだけを、試料から入射側へ距離 `L` のスクリーン平面に投影します。

回転角は、入力した入射方向とスクリーン座標を `a軸 -> b軸 -> c軸` の順に回転して適用します。c 軸入射時の c 軸回転は入射方向そのものは変えませんが、スクリーン上のスポット配列を回転させます。
