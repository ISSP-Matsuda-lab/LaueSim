# LaueSim

X 線を入射し、入射側に戻ってくる後方反射 Laue スポットをブラウザ上でシミュレートする単体HTMLアプリです。

## 使い方

`index.html` をブラウザで開いてください。依存パッケージやビルド手順は不要です。

入力できる条件:

- スクリーンまでの距離 `L` mm
- 格子定数 `a,b,c` Å と格子角 `alpha,beta,gamma` deg
- 入射方向 `[u v w]`
- 入射方向/試料座標の回転角 `a軸,b軸,c軸` deg
- X線スペクトルの波長範囲
- 探索する最大ミラー指数

グラフ操作:

- ドラッグ: 表示範囲を移動
- マウスホイール/トラックパッドスクロール: 拡大縮小
- 照準ボタン: 表示をリセット

現在のモデルは任意の格子角を持つ一般セルの後方反射 Laue です。初期値は `a=b=3.98 Å, c=12.3 Å, alpha=beta=gamma=90 deg, L=50 mm, c軸回転=45 deg`、入射方向 `[0 0 1]` で、 c 軸入射を想定しています。

実格子ベクトル `a,b,c` から逆格子基底 `a*,b*,c*` を外積で作り、各逆格子ベクトル `h a* + k b* + l c*` に対して Laue 条件を満たす波長を計算します。回折方向が入射方向と反対側へ戻るスポットだけを、試料から入射側へ距離 `L` のスクリーン平面に投影します。

回転角は、入力した入射方向とスクリーン座標を結晶の `a軸 -> b軸 -> c軸` の順に回転して適用します。c 軸入射時の c 軸回転は入射方向そのものは変えませんが、スクリーン上のスポット配列を回転させます。

## Vercel へのデプロイ

このアプリはビルド不要の静的サイトです。Vercel では Framework Preset を `Other`、Output Directory をリポジトリルート `.` として配信します。設定は `vercel.json` に含めています。

初回のみ Vercel プロジェクトにリンクします。

```bash
npx --yes vercel@latest link
```

プレビュー環境にデプロイします。

```bash
npm run deploy
```

本番環境にデプロイします。

```bash
npm run deploy:prod
```

デプロイ前の簡易チェックは次のコマンドで実行できます。

```bash
npm run check
```
